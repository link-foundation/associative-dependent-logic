# Rocq Export

`rml export rocq` translates the typed LiNo fragment into Rocq source.
The exporter is intentionally conservative: it accepts only forms with a
direct Rocq counterpart and rejects probabilistic, many-valued, and
runtime-operator features.

## CLI

JavaScript:

```sh
cd js
node src/rml-links.mjs export rocq ../examples/dependent-types.lino -o dependent_types.v
```

Rust:

```sh
cd rust
cargo run -- export rocq ../examples/dependent-types.lino -o dependent_types.v
```

When `-o` or `--output` is omitted, the generated Rocq source is printed to
standard output. If you plan to compile the result with `rocq compile`, choose
a `.v` filename whose basename is a valid Rocq module identifier, for example
`dependent_types.v` rather than `dependent-types.v`.

## Supported Subset

| LiNo form | Rocq output |
|-----------|-------------|
| `(A: (Type 0) A)` | `Parameter A : Set.` |
| `(A: Type A)` | `Parameter A : Type.` |
| `(x: A x)` | `Parameter x : A.` |
| `(f: (Pi (A x) B))` | `Parameter f : forall x : A, B.` |
| `(f: lambda (A x) body)` | `Definition f := fun x : A => body.` |
| `(apply f x)` and `(f x)` | `(f x)` |
| `(Pi (A x) B)` and `(forall (A x) B)` | `forall x : A, B` |
| `(a = b)` | `a = b` |
| `(? (x of A))` | `Check (x : A).` |
| `(? (type of x))` | `Check x.` |
| `(inductive T (constructor c) ...)` | `Inductive T : Set := ...` |

The exporter sanitizes LiNo names into Rocq identifiers by replacing
non-identifier characters with `_` and reports an error if two distinct LiNo
names would collide after sanitization.

## Example

Input:

```lino
(Natural: (Type 0) Natural)
(zero: Natural zero)
(succ: (Pi (Natural n) Natural))
(identity: lambda (Natural x) x)

(? (zero of Natural))
(? (type of zero))
(? (apply succ zero))
```

Output:

```coq
(* Generated by rml export rocq. *)

Set Universe Polymorphism.

Parameter Natural : Set.
Parameter zero : Natural.
Parameter succ : forall n : Natural, Natural.
Definition identity := fun x : Natural => x.
Check (zero : Natural).
Check zero.
Check (succ zero).
```

## Inductives

Input:

```lino
(inductive Natural
  (constructor zero)
  (constructor (succ (Pi (Natural n) Natural))))
```

Output:

```coq
(* Generated by rml export rocq. *)

Set Universe Polymorphism.

Inductive Natural : Set :=
| zero : Natural
| succ : forall n : Natural, Natural.
```

RML-generated eliminator names such as `Natural-rec` are not emitted because
Rocq generates its own eliminators for `Inductive` declarations.

## Rejected Forms

The Rocq exporter rejects forms that depend on RML's probabilistic evaluator
or metatheory machinery:

- Probabilistic assignments such as `((p = true) has probability 1)`
- Runtime operator and configuration forms such as `(and: min)`,
  `(or: max)`, `(range: 0 1)`, and `(valence: 2)`
- Untyped term declarations such as `(p: p is p)`
- Numeric literals and arithmetic expressions
- Imports, namespaces, relations, mode/world declarations, totality,
  coverage, termination, tactic, and coinductive forms
- Self-referential `(Type: Type Type)`. Use stratified `(Type N)` universes
  for Rocq export.
